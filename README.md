Trading Hub
Personal finance and trading dashboard using FastAPI + React + Postgres + Docker

Stack:
Backend: FastAPI, SQLAlchemy, Alembic
Tasks: Celery (Redis broker), fetch jobs
Database: Postgres
Frontend: React + TypeScript
